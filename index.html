PHPCron Runner 1.0 : run your php in background

Requirement
  • PHP Verion >= 5.0
  • OS = linux/window
  • Webserver = Apache/IIS/ngix
Dependancy
  • nohup for linux
  • COM extension for winodw
  • Webserver = Apache/IIS/ngix
Folder structre
  • [+].classes ( user defined classess)
  • [+].configs ( user configuration classess)
  • [+].systems ( Lib lxasses for cron runner )
  • Webserver = Apache/IIS/ngix
Basic Setting
  • define your php path for winodw i,e ex
    define("CRON_PHP_PATH_WIN", 'D:/xampp/php/php.exe'); define your php path for linux i,e ex
    define("CRON_PHP_PATH_LINUX", '/usr/bin/nohup /usr/local/bin/php');
  • COM extension for winodw
  • Webserver = Apache/IIS/ngix
